---- js directory ----

This directory contains the JavaScript utilized by various pages, such as index.php and restaurants.php

-all.js: utilized by fontawesome
-boostrap. ...: bootstrap code for displaying the stars in restpage
-indexSearch.js: stores the search from home page search bar
-jquery-3.5.1.js: jQuery
-rating.js: utilized to get the rating from a new review on restpage
-restaurants.json: array of all restaurants and their data
-restpage.js: scripts for displaying a single, predetermined restaurant to restpage
-search.js: scripts for searching restaurants on the restaurants page
-upload-display.js: helps display gallery images
